Business code courses

Angular 4

Aprende a desarrollar aplicaciones web modernas desde cero, basadas en el modelo MVC, con Angular 4, un framework de Javascript creado y mantenido por Google.

Metodologías Scrum

Mejora tu flexibilidad y productividad, con metodologías scrum, totalmente enfocadas al trabajo en equipo y a obtener el mejor resultado posible de un proyecto.

Servicios REST con Django

Consigue crear sitios web rápidamente con Django, un framework de desarrollo web, escrito en Python y basado en el modelo MVC, poniendo énfasis en el principio DRY "Dont repeat yourself".

Git: Introducción y flujos de trabajo

Entiende en implementa Git en tu flujo de trabajo, el software de control de versiones más popular del mundo.

Xamarin

Desarrolla aplicaciones móviles nativas una sola vez y haz que funcionen en cualquier plataforma sin duplicar código con Xamarin, plataforma de desarrollo de Microsoft.

React + Redux

Aprende a trabajar con React.js y Redux para construir aplicaciones web modernas y rápidas. Su propósito es ayudar a los desarrolladores a crear Single Page Applications con JavaScript.

Reactnative

Si conoces Javascript y React, aplica estos conocimientos para crear una app de iOs y Android reutilizando el mismo código, manteniendo los componentes nativos de cada plataforma.

Scala Introduction

Adéntrate de lleno en la programación funcional con Scala, un lenguaje multiparadigma y con una sintaxis concisa, elegante y flexible que minimiza la sintaxis innecesaria.

Spark para Scala

Conoce Spark, un marco de procesamiento para macrodatos de código abierto escrito en Scala, que cada vez es más conocido, especialmente en escenarios de aprendizaje automático.

Desarrollo de APIs en node.js (usando Express y Mongo)

Crea APIS de una forma sólida, rápida y sencilla con Express, un framework de Nodejs, que tiene miles de métodos de programa de utilidad HTTP y middleware a tu disposición.

Testing de aplicaciones Javascript

Esta formación va dirigida a analistas e ingenieros de pruebas y de desarrollo software, así como a programadores y tecnicos de sistemas, que quieran desarrollar la capacidad de automatizar pruebas.

Aplicaciones cross-platform con Electron

Conoce Electron, un framework para JavaScript desarrollado por GitHub, que permite el desarrollo de aplicaciones enriquecidas de escritorio mediante el uso de tecnologías web.

Vuejs

Vue.js es uno de los frameworks que está creciendo exponencialmente en el mundo de la programación frontend, especializado en la capa de vista. Es ideal para construir interfaces web interactivas.

Devops con AWS

Conoce la nube con Amazon Web Services, una colección de servicios de computación en la nube, que en conjunto forman una plataforma de computación en la nube.

Docker

Aprende a automatizar tus despliegues con Docker, un proyecto de código abierto que automatiza el despliegue de aplicaciones dentro de contenedores de software, basado en Linux.

Ansible

Aprende a configurar y automatizar cualquier tipo de tarea en el ámbito de administración de sistemas con Ansible, plataforma libre que además, es una herramienta de implementación y orquestación.

Desarrollo de videojuegos con Unity 3d

Aprende a desarrollar videojuegos de una forma fácil y rápida con Unity 3d, motor de videojuegos multiplataforma, compatible con el desarrollo tanto 2D como 3D.

Desarrollo de videojuegos con Unreal Engine

Aprende a desarrollar videojuegos con unreal engine, motor de videojuegos de PC y consolas, que ofrece todo lo necesario para hacer desde juegos 2D hasta juegos catalogados triple A.

Go programming language

Conoce Go, un lenguaje de programación compilado, concurrente, imperativo, estructurado, no orientado a objetos y con recolector de basura, desarrollado por Google.

Machine Learning y Data Science con Scikit-Learn y Psypark

Conoce el Machine Learning, una rama de la inteligencia artificial, con Scikit-Learn y Psypark, librerías de machine learning para Python .

Criptografía: Algoritmos

Mejora la seguridad de tus aplicaciones desarrollando algoritmos criptográficos, añadiendo características como autenticación, integridad y confidencialidad a los datos.

Desarrollo de webs estáticas con Hugo

Desarrolla webs estáticas con Hugo, un framework que proporciona herramientas esenciales al flujo de trabajo de un sitio web estático, permitiendo construir sitios webs rápidamente y creativos.

Metaheurísticas

Aprende a resolver cualquier tipo de problema computacional general, usando los parámetros dados por el usuario, de una forma más eficiente y óptima, con el método heurístico.

Category Theory en Scala

Aprende a aplicar en tus desarrollos la teoría de categorías y a cumplir las propiedades algebráicas de una categoría con Scala.

Programación en R para ciencia de datos

Adéntrate en el mundo del Big Data con R, un lenguaje de programación interpretado y diseñado específicamente para el análisis estadístico y la manipulación de datos.
